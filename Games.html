Flappy Bat is a clone app of the popular game Flappy Bird. It has Flappy Bird's aesthetics and same addictive qualities but with a few differences. You are able to win ribbons based on the achievement you've accomplished. Flappy Bat stays simple and to the true nature of the original game. Click here to see a video preview of Flappy Bat.

This game was created using Xcode with Swift as its programing language. The game also uses core data as the current database for highscores. It uses a few core frameworks embedded in Xcode along with Google's AdMob library for advertisement placement and presentation. The beautiful graphics were created by Jon W.
