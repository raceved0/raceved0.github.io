  • Augmented Reality

This augmented reality application will make you feel like you're in the future. Unlike virtual reality, which puts the user inside the application, Augmented Reality helps the application become part of the real world. The camera software scans the current frame for a predetermined black and white pattern. Once the pattern is found, the software will replace the pattern with a photo or video. Though this program only performs simple operations, the bigger intent is to create a full application that interacts with the user in a real-time environment. For a better understand of how augmented reality works, click here and here. Augmented Reality was created using Microsoft's Visual Basic with Open CV C++ library.

  • Motion Detection

Motion Detection is a camera-based program that tracks an objects movement. The software starts recording if an object either moves within the frame or moves into the frame. If the object stops moving inside the frame or leaves the frame for six seconds, this program will stop recording. The UI is still a work in progress, and there are many features still not implemented into the program. Motion Detection was created using Microsoft's Visual Basic with Open CV C++ library.

  • Object Detection

This object detection program tracks objects based on their hue, saturation, and value. The user takes a photo of the object they would like to detect. The program then inputs the values into the algorithm and starts scanning for this color. Once this object enters frame, the camera will start recording. To stop recording, the object needs to leave the frame or user clicks on the stop button. Object Detection was created using Microsoft's Visual Basic with Open CV C++ library.
