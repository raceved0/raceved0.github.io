This website was written completely from scratch and used only a few resources to create it. Also, no templates, free website creators, or code from other developers have been used to develop this website. Though the process of using original material was time consuming and in some cases frustrating, the knowledge gained from it is immeasurable. The only program used for this website was Brackets for coding the site.

The programing languages used in this project are CSS, JavaScript, jQuery, and HTML. All the code is original without the help from other developers. From this came many bugs, and when dealing with JavaScript, the issues with browser incompatibilities.

In the Home page, quotes from influential computer scientist and innovators in the field are cycled through using Javascript. Also, the parallax effect in the Projects and Games pages had been the the most difficult part of creating this site. The functionality of all site features are original code.

On the top left corner of this website, there is the Code.org logo. The logo is JavaScript interactive and if the user hovers over the letters in the logo, they will transform to different colors. I participated in Code.org's Hour of Coding. I am very passionate about Computer Science and its role in public schools. With the support from websites like Code.org, Computer Science can have its own curriculum in schools and thus creating a more technical mindset for developing minds while still being fun and interactive. Simple ideas like these help adjust our youths to a fast expanding field.
